# 地理 因果関係ツリー

高校地理の学習向けに、**気候 → 植生 → 農業**の流れをたどる静的Webアプリです。  
低緯度ルートには **Aw（サバナ気候）** と **Am（熱帯モンスーン気候）** の枝を追加しています。

## 使い方

1. `index.html` を開くと、最初に「低緯度 / 中緯度 / 高緯度」が表示されます。
2. 中央の候補カードから1つ選ぶと、候補は次の段階だけに更新されます。
3. 上部には「選んだ流れ」のみが残ります。
4. 右側（スマホでは下部）に、**今選んでいる要素の説明**が1件だけ表示されます。
5. `1つ戻る` で直前の選択を戻し、`リセット` で最初からやり直せます。

## GitHub Pages で公開する手順

1. このリポジトリを GitHub に push します。
2. GitHub の **Settings → Pages** を開きます。
3. **Build and deployment** の Source で `Deploy from a branch` を選びます。
4. Branch を `main`（または公開したいブランチ）、Folder を `/ (root)` に設定して保存します。
5. 数分後に表示される URL でスマホからアクセスできます。

## ファイル構成

- `index.html` : アプリ本体の構造
- `style.css` : ダークテーマとレスポンシブレイアウト
- `script.js` : 因果関係データと画面更新ロジック
- `manifest.webmanifest` : PWA向けメタ情報
- `icons/` : アイコン（プレースホルダSVG）
